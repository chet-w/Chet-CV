KPSmart Mail Delivery System

Kelburn Post Smart (KPS) was a project completed over eight weeks in the first half of 2015. It is a web-app used to send mail and manage the workings of a fictional mail service, including features such as updating cost-price information, and viewing accurate real-time statistics from the application.

The application is built on a HTML5 backbone, published in Jade, while using JavaScript to communicate to a backend SQL database. Working with four other students for this project, I worked as chief front end developer for this project creating all the visual elements on the interface, and was second in command for management, stepping in for our Projet Leader when he was unavailable.
